= Basic Information =

私の夢は将来一軒家のマイホームを建てることだ。
これからマイホームを彩る理想のアイテムを紹介する。家の立地条件としては、周りに豊かな自然が広がること。 しかし、スーパーは車で15分以内が条件だ。ニュートラルな田舎を知っている人がいたら教えてください。

= EXTERIOR =

外装は温かみのある雰囲気を持たせたい。 壁はクリームホワイト、屋根は赤煉瓦、そして開放感味わえる大きな窓

= INTERIOR =

内装はウッディーベースで至る所にお気に入りのカラーを加えたい。そして、キッチンにはスパイスを大量に揃えたい。

= GARDEN =

家のすぐ近くに川を流す。庭ではちょっとした家庭菜園を行い、野菜を育てたい。そしてハンモック。
